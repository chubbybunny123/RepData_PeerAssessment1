Imputing missing values

  1. Calculate and report the total number of missing values in the dataset (i.e. the total number of rows with NAs)

There are 2304 rows with NA in the steps column.

  1. Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc.

I will fill the NA's with the mean for that 5-minute interval. We've already computed the average steps in each interval for the previous problem, so whenever there's an NA, copy over the average value.

  1. Create a new dataset that is equal to the original dataset but with the missing data filled in.

Let's add this as a new column in the data table, with name “newSteps”.

for(i in 1:length(DT$interval)) {
    if(DT$isNA[i]) {
        idx <- which(avgDay$interval == DT$interval[i])
        DT$newSteps[i] <- avgDay$avgsteps[idx]
    } else {
        DT$newSteps[i] <- DT$steps[i]        
    }
}
